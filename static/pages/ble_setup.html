蓝牙配网（Web Bluetooth）

实体 ESP32-WROOM-32 到货后，使用本页在 Chrome / Edge 桌面版 完成：扫描 → 连接 → 下发 WiFi 账号密码 → 观察开发板 LED 与串口日志。

流程说明

  1. 开发板上电，固件进入 BLE 广播状态（具体服务 UUID 以 firmware/ 内代码为准，后续补充）。
  2. 本页点击「扫描并连接」，浏览器弹出设备选择框，选中你的 ESP32。
  3. 填写家庭/实验室 WiFi 的 SSID 与密码，写入特征值；设备断开 BLE 后自动连接 WiFi 并向平台注册。
  4. 回到「设备管理」绑定或核对设备是否出现在列表中。

当前无硬件：下方按钮仅演示交互与异常提示，不会发起真实蓝牙连接。

浏览器支持

Web Bluetooth 在多数手机浏览器上受限，请优先使用 Windows 桌面 Chrome。若 navigator.bluetooth 不存在，页面会在真机联调时给出提示。
