蓝牙配网（Web Bluetooth）

使用本页在 Chrome / Edge 桌面版 完成：扫描 → 连接 → 下发 WiFi JSON → 观察设备回执与串口日志（推荐 115200）。

流程说明

  1. 开发板上电，固件进入 BLE 广播状态（本页已与固件 UUID 对齐）。
  2. 本页点击「扫描并连接」，浏览器弹出设备选择框，选中你的 ESP32。
  3. 填写 WiFi 的 SSID 与密码（可选填写服务器地址与密钥），点击「下发配置」写入特征值。
  4. 设备将写入 NVS 并尝试连接 WiFi → 调用 POST /api/device/register 注册/校验；本页会显示设备通过 BLE 通知回传的进度。
  5. 回到「设备管理」绑定或核对设备是否出现在列表中。

提示：Web Bluetooth 仅在安全上下文可用（HTTPS 或 localhost）。

配网操作区

浏览器支持

Web Bluetooth 在多数手机浏览器上受限，请优先使用 Windows 桌面 Chrome。若 navigator.bluetooth 不存在，页面会在真机联调时给出提示。
