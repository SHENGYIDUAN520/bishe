## 3. 建议公式清单

### 公式 3-1 平均温度计算公式

```text
T_avg = (1/n) * Σ(T_i)
```

说明：用于表示某时间窗口内温度平均值。

### 公式 3-2 平均湿度计算公式

```text
H_avg = (1/n) * Σ(H_i)
```

说明：用于表示某时间窗口内湿度平均值。

### 公式 3-3 阈值告警判断公式

```text
Alarm = 1, 当 T > T_threshold
Alarm = 0, 当 T <= T_threshold
```

说明：用于表示温度是否触发告警。

### 公式 3-4 冷却时间判断公式

```text
Send = 1, 当 (t_now - t_last) >= Δt
Send = 0, 当 (t_now - t_last) < Δt
```

说明：用于表示在冷却窗口内是否允许再次发送告警邮件。
